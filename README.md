# smcss

A way to organize and write CSS https://smcss.vbarbarosh.com

## Installation

    $ npm i vbarbarosh/smcss

## Before begin

    $ bin/configure
    $ bin/build
    $ bin/test

## A few examples

    <div class="abs-f ma hsplit w800 h200 bs10 rainbow">
        <div class="w200"></div>
        <div class="fluid"></div>
        <div class="w200"></div>
    </div>
