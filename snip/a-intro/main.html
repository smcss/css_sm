Introduction

Explain names — the rule of thumb for choosing names; units — a convention for specifying numbers; hover — a convention for classes targeting :hover state; child — a convention for classes targeting immediate children; recursive — a convention for classes targeting descendants.

Numbers

Number without any suffix represent pixels

Number with m suffix — represent 1/10th of a em, e.g. 05m is 0.5em, 10m is 1.0em, etc.

Number with p suffix — represent percents, e.g. 2p is 2%, 10p is 10%

Fractions

Fractions are specified with f letter between whole and fraction parts, e.g. 2f3 is 2.3 pixels; 2f3m is 2.3em, 2f3p is 2.3%

Unitless numbers

By default numbers without prefix denote pixels. Exclusion of this rule is opacity, z-index, ...

Negative Numbers

Suffix n means that number is negative.

All negative numbers should have suffix n.
