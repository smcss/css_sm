Grid

Grid is a way to split whole space into several parts. The general syntax provided below, where M — specifies how many parts a whole should be divided by. N — specifies how many parts a column will take.

<div class="gridN">
    <div class="colM">...</div>
    <div class="colM">...</div>
    ...
    <div class="colM">...</div>
</div>

Note that grid provides only a way to distribute widths. And it always should be used with something like .flex or .child-fl.

<div class="grid4 flex-rows">
    <div class="colM">...</div>
    <div class="colM">...</div>
    ...
    <div class="colM">...</div>
</div>
